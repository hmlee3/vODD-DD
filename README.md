# vODD-DD: Visual Documentation Protocol for Data-Driven Agent-Based Models

A one-page visual documentation template for data-driven agent-based models (DDABMs),
extending the ODD protocol with explicit data-driven components.

## What is vODD-DD?

vODD-DD (visual ODD for Data-Driven models) is a visual documentation protocol 
designed to support transparent and reproducible reporting of DDABMs. It extends 
the standard ODD protocol by explicitly representing data sources, data pipelines, 
calibration and validation procedures, and data–model component mappings in a 
single scannable diagram. 

## Repository Contents

- `/template` — PowerPoint template (.pptx) with instructions
- `/example` — Completed SIR epidemiological case study example
- `/library` — Python library for automated diagram generation *(under development)*
- - `/literature-review`— Full list of 67 records retrieved from Google Scholar

## How to Use

1. Download the PowerPoint template from `/template`
2. Follow the instructions on the How to Use slide
3. Fill in each component with your model information
4. Export as PDF, PNG, or keep as .pptx
